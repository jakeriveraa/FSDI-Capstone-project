About Capstone Project

A capstone project showcasing full-stack web development skills

Welcome to My Capstone Project, a full-stack web development capstone course project. This project demonstrates skills in Django, HTML, CSS, JavaScript, and Bootstrap by creating a functional, responsive web application with user-friendly features.

Project Purpose

The purpose of this Capstone Project is to simulate an online shopping experience, complete with a homepage, product sections, mini-games, and an interactive UI. This project helps showcase practical full-stack development skills in a real-world scenario.

Future Enhancements

Future iterations could include user accounts, shopping carts, payment simulation, more mini-games, and advanced product filtering.
